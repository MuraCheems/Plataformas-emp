Acerca del creador
Estudiante de ingenieria de sistemas de 5to semestre, realizo esta pagina web para entregar de primer ejercicio de programacion en plataformas empresariales.
CSS
Este fue usado para darle estetica al sitio, desde algunas animaciones hasta los fondos de esta misma.
HTML5
Usamos HTML5, lenguage markup, para definir cada uno de los elementos que contendra cada pagina web
Laragon
El localhost de Laragon fue usado para simular el hosting de la pagina localmente.
Javascript
Lenguage que se uso en algunas animaciones y en la parte del localstorage para almacenar usuarios y simular un login, registro, un recordar contresena y una redireccion de paginas
